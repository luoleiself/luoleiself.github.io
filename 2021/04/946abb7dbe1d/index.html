Git命令其他篇

发表于 2021-04-20 更新于 2024-07-26 分类于 tools 1k 1 分钟

其他篇

submodule 仓库

submodule init 初始化
submodule deinit 删除
submodule set-branch 设置分支

设置嵌套仓库的默认远程关联分支

submodule set-url 设置地址

设置嵌套仓库新的地址,会自动同步新的地址配置项

submodule add 添加

在当前指定目录下添加仓库

status|summary 查看

  • 递归查看嵌套仓库的状态

  • 查看嵌套仓库的提交记录

submodule update 更新

递归更新嵌套仓库信息

archive 归档

  • list: 显示支持的归档文件格式

  • format: 指定归档文件格式

  • prefix: 指定归档文件前缀

  • output|o: 将归档文件写入到指定文件非输出流

  • add-file: 添加未被追踪的文件到归档文件中

clean 清除

  • n|dry-run: 不移除任何东西,只显示会做什么
  • f|force: 强制删除
  • i|interactive: 交互式操作
  • e: 使用正则表达式匹配
从工作区移除未被追踪的文件
